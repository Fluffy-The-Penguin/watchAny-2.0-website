Instant Torrent Streaming

FAQ

Common questions.
Quick answers.

Is watchAny completely free?↓
Yes — zero cost, zero ads, zero subscriptions. It's fully open source under the MIT license. The source code is hosted on GitHub.
What can it actually play?↓
Virtually anything: local MKV/MP4 files, magnet links (torrents) using built-in TorrServer, direct video URL streams, HLS/DASH airings, and customized extensions via HStream.
Does it support 4K and HDR playback?↓
Yes. Under the hood, watchAny features hardware-accelerated playback via libmpv for lag-free 4K, HDR, and HEVC video streams.
Do I need an account to use it?↓
No account required. Connect an AniList profile to synchronize your watch logs and progress, but it is 100% optional.
